# Plugin: Bookmarks (`bookmarks`)

[← Katalog](README.md)

## Deutsch

| Feature | Beschreibung |
|---------|----------------|
| Gruppen | Mehrere Gruppen, einzeln ausblendbar (👁️) |
| Icons | Emoji oder PNG/JPG |
| Drag & Drop | Sortieren innerhalb und zwischen Gruppen |
| Layout | Raster oder waagerechte Scroll-Zeile |
| Kachelbreite | Min/Max px, optional feste Spaltenbreite |
| Neuer Tab | Pro Link einstellbar |

## English

Quick links with collapsible groups, custom icons, drag & drop, grid or horizontal row layout.
