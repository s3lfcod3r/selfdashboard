# Plugin: Selfstream (`selfstream`)

[← Katalog](README.md)

## Deutsch

Zeigt **aktive IPTV-Streams** aus einer [Selfstream](https://github.com/kabelsalatundklartext/selfstream)-Admin-Instanz.

| Thema | Details |
|-------|---------|
| **API** | `POST /api/selfstream` |
| **Einstellungen** | Basis-URL (ohne `/admin`), Admin-Passwort, Intervall |
| **Netz** | Selfstream muss vom Container aus erreichbar sein (LAN) |

## English

Live streams from Selfstream admin API. Set base URL and admin password in widget settings.
