SelfDashboard + Unraid: Netz-JSON — einfacher Weg (ohne Nginx-Login)
================================================================

Problem: `https://DEIN-TOWER/net-stats.json` → Login-HTML (302), kein JSON.

Lösung in 3 kurzen Schritten:

1) Ordner auf dem Tower (einmalig, SSH):

   mkdir -p /mnt/user/selfdashboard-stats

2) Kleinen Datei-Server starten (einmalig nach jedem Reboot, oder per User Script „Array Start“):

   cd /mnt/user/selfdashboard-stats
   nohup python3 -m http.server 8765 --bind 0.0.0.0 >/dev/null 2>&1 &

   Test im Browser:  http://DEINE-TOWER-IP:8765/net-stats.json
   → muss eine JSON-Zeile zeigen (nicht die Unraid-Login-Seite).

3) Dein Script, das net-stats.json schreibt, legt die Datei HIER ab:

   OUT_FILE=/mnt/user/selfdashboard-stats/net-stats.json

   Im SelfDashboard-Unraid-Widget bei „JSON-URL“ dann eintragen:

   http://DEINE-TOWER-IP:8765/net-stats.json

SelfDashboard 1.3.3+ lädt diese URL serverseitig über /api/net-stats-proxy
(kein CORS im Browser). Nur private IPv4-Adressen sind erlaubt.

Hinweis: SelfDashboard (Next.js) muss den Tower im LAN erreichen können
(z. B. SelfDashboard im gleichen Heimnetz oder im Docker-Bridge mit LAN-Zugang).
